# Veronika

A small, elegant Python backend starter that combines Django and FastAPI for different needs: rapid development and lightweight APIs.

Why Veronika?
- Simple and clear structure to get you started fast.
- Use Django for full-featured web apps and FastAPI for fast, async APIs.

Features
- Ready-to-adapt project layout for Django and FastAPI components.
- Example scripts to run services locally.
- Minimal, readable code so you can focus on features.

Requirements
- Python 3.9+
- pip

Getting started
1. Clone the repository:

   git clone https://github.com/azarmi-dev/Veronika.git
   cd Veronika

2. Create a virtual environment and install dependencies:

   python -m venv .venv
   source .venv/bin/activate  # macOS / Linux
   .venv\Scripts\activate     # Windows
   pip install -r requirements.txt

Run locally
- For a quick test, run the simple script in main.py:

   python main.py

- For Django or FastAPI apps, follow the respective README or app folders (add details here as you scaffold the apps).
